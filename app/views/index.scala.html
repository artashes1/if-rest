@() Play REST API

Play REST API

This is a placeholder page to describe the REST API.

To query all users, you can do a get:

			http GET {host}/api/v1/users

To create new users, do a post

			http POST {host}/api/v1/users

To get user by ID, do a get

			http GET {host}/api/v1/users/{userId}

To update existing user, do a put

			http PUT {host}/api/v1/users/{userId}

To query all roles, you can do a get:

			http GET {host}/api/v1/roles
